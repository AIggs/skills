# Project Homepage Generator

Generate a stunning project homepage with high-tech dark theme, swirl particle background, rainbow gradient effects, and rich animations.

## Input Parameters

When user provides:
- **Project name**: e.g., "AIggs", "TechAI", etc.
- **Header menu items**: Array of navigation links
- **Hero section**: Badge text, title (with rainbow keywords), subtitle, CTA buttons
- **Platform/Features section**: Array of feature cards with icon, title, description, tag
- **How It Works section**: Array of steps with number, icon, title, description
- **Code Preview section**: Code snippet with syntax highlighting
- **Use Cases section**: Array of use case cards
- **Stats/Coverage section**: Array of statistics
- **About section**: Array of info cards
- **Contact section**: Title, description, CTA
- **Footer**: Brand tagline, links columns, social icons

## Output

Complete HTML file with all the stunning effects pre-configured.

## Features Included

### 1. Swirl Ambient Canvas Background
- Codrops-inspired particle swirl effect
- Uses Simplex Noise for fluid movement
- 60 particles by default, subtle glow
- Configurable opacity and brightness

### 2. Rainbow Gradient Text
- Animated color shift effect
- Keywords highlighted with `.rainbow` class
- Smooth 3s animation cycle

### 3. Rich Entrance Animations
- `fadeInUp` - Default upward animation
- `fadeInLeft` / `fadeInRight` - Slide animations
- `fadeInScale` - Zoom effect
- `flipIn` - 3D flip for cards
- Staggered delays (0.1s - 0.6s)

### 4. Hover Interactions
- Card lift with shadow
- Icon scale and rotate
- Border glow effects
- Step number rotation

### 5. SVG Icons
- Flat design, consistent style
- Accent color fill (#00d4d4)
- Hover glow effect

### 6. Typography
- Font: Outfit (ultra-light 200 weight available)
- JetBrains Mono for code and technical labels
- Clean hierarchy with consistent spacing

### 7. Color Scheme
- Primary: #0a0a0c (deep dark)
- Secondary: #0f0f12 (slightly lighter)
- Accent: #00d4d4 (cyan)
- Text: #f5f5f7 (white), rgba for muted

## Key CSS Classes Reference

| Class | Effect |
|-------|--------|
| `.rainbow` | Rainbow gradient animated text |
| `.animate` | Fade in up animation |
| `.animate-left` | Fade in from left |
| `.animate-right` | Fade in from right |
| `.animate-scale` | Scale up animation |
| `.animate-flip` | 3D flip animation |
| `.delay-1` to `.delay-6` | Animation delay stagger |
| `.platform-card` | Feature card with hover lift |
| `.step-number` | Step indicator with rotation on hover |
| `.use-case-card` | Use case card with icon scale |
| `.coverage-stat` | Statistics with glow effect |
| `.about-card` | Info card with slide effect |

## Responsive Breakpoints

- Desktop: 1024px+ (full layout)
- Tablet: 768px-1024px (2-column grids)
- Mobile: <768px (single column, hidden nav)

## Customization Tips

1. **Adjust particle count**: Change `particleCount` in JS (default 60)
2. **Change accent color**: Update `--accent` in CSS (default #00d4d4)
3. **Add more rainbow words**: Wrap text in `<span class="rainbow">word</span>`
4. **Adjust animation speed**: Modify `.animate` duration (default 0.6s)
5. **Change background opacity**: Adjust `.content--canvas canvas` opacity (default 0.25)
