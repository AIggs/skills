# Aiggs Skills

---

## Available Skills

### 🔧 fe-homepage

Generate stunning project homepages with AI-powered design.

```bash
npx skills add https://github.com/AIggs/skills --skill fe-homepage
```

### 📸 chrome-cap-extension

A Chrome extension that allows users to capture and save screenshots of web pages. It provides an easy way to capture the entire page, a specific area, or the visible portion of the page. Users can also annotate and edit the captured screenshots before saving or sharing them. The extension is designed to enhance productivity and streamline the process of capturing and managing web content visually.

**Features:**

- 🌐 Capture entire page, specific area, or visible portion
- ✏️ Annotate and edit screenshots
- 💾 Save or share with ease

## Usage

```bash
# Add a skill
npx skills add https://github.com/AIggs/skills --skill <skill-name>

# List available skills
npx skills list
```
